# Diplom_1
## **Юнит-тесты**
_веб-приложение: Stellar Burgers_

### В проекте:

#### 1. Реализованы тесты на методы класса Burger:

* параметризированный тест на перемещение ингредиентов в бургере
* параметризированный тест на получение цены бургера с разным наполнением
* параметризированный тест на получение чека бургера с разным наполнением
* позитивные и негативные тесты на добавление ингредиентов в бургер
* позитивные и негативные тесты на установку булочки в бургер
* позитивные и негативные тесты на удаление ингредиентов в бургере
* негативные тесты на перемещение ингредиентов в бургере
* негативные тесты на получение цены бургера
* негативные тесты на получение чека бургера

#### 2. Использованы технологии:

* Java 11
* Maven 4.0.0
* JUnit 4.13.2
* Assertj 3.27.6
* Mockito 3.12.4
* Jacoco 0.8.7

### Как запустить тесты
`mvn clean test`
